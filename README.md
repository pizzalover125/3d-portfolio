# 3D Portfolio Website

a three-dimensional portfolio website. minimalist. jaw-dropping. awesome.

### technical information

made with AFrame and HTML, CSS, JS. AFrame runs on three.js. 

### features
- 4 panels that look like glass
- a bunch of boxes of text
- full player controls, allowing used to move around

### notes
it looks pretty simple but its a lot harder than it looks. Aframe is awesome but it doesn't come with much. for example, the background looks simple, but in reality it is a shader that took a LOOOOOOONG time to figure out. each block of text had to be placed precisely. overall, this was an awesome experience building! I tried adding 3d models but there was a bunch of browser incompatability and it always just glitched out.
